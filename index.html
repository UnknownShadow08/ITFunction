1 · Domain & Range
Function
Domain → Range · Use case
sin⁻¹(x)
|x| ≤ 1 → [−π/2, π/2] · Smallest angle whose sin is x
cos⁻¹(x)
|x| ≤ 1 → [0, π] · Smallest non-negative angle whose cos is x
tan⁻¹(x)
x ∈ ℝ → (−π/2, π/2) · Defined for all reals
cot⁻¹(x)
x ∈ ℝ → (0, π) · Always gives angles in (0,π)
sec⁻¹(x)
|x| ≥ 1 → [0,π/2)∪(π/2,π] · Excludes π/2
cosec⁻¹(x)
|x| ≥ 1 → [−π/2,0)∪(0,π/2] · Excludes 0
★ x > 0 → all 6 ITFs give acute angles
★ x < 0 → sin⁻¹, tan⁻¹, cosec⁻¹ give IVth quadrant (−π/2 to 0)
★ x < 0 → cos⁻¹, cot⁻¹, sec⁻¹ give IInd quadrant (obtuse angle)
★ IIIrd quadrant is NEVER used in range of any ITF

3 · Property 2 · f⁻¹(f(x)) [Inner cancels Outer — with period]
sin⁻¹(sin x) — period 2π, odd function
x ∈ [−π/2, π/2]
= x · Principal branch — direct
x ∈ [π/2, π]
= π − x · Reflect from π/2
x ∈ [−π, −π/2]
= −π − x · Reflect from −π/2
cos⁻¹(cos x) — period 2π, even function
x ∈ [0, π]
= x · Principal branch
x ∈ [−π, 0]
= −x · Flip for negative
tan⁻¹(tan x) — period π, odd function
x ∈ (−π/2, π/2)
= x · Principal branch
x ∈ (π/2, 3π/2)
= x − π · Subtract period
x ∈ (−3π/2, −π/2)
= x + π · Add period
cot⁻¹(cot x) — period π, neither even nor odd
x ∈ (0, π)
= x · Principal branch
x ∈ (−π, 0)
= x + π
x ∈ (π, 2π)
= x − π
★ sec⁻¹(sec x) — period 2π, even function
★ cosec⁻¹(cosec x) — period 2π, odd function
★ KEY TRICK: For f⁻¹(f(x)) — check if x lies in principal range. If not, shift by period to bring it in range.
4 · Property 3 · Complementary Pairs [Sum = π/2]
sin⁻¹x + cos⁻¹x = π/2
|x| ≤ 1 · Convert sin⁻¹ ↔ cos⁻¹
tan⁻¹x + cot⁻¹x = π/2
x ∈ ℝ · Very frequently used
sec⁻¹x + cosec⁻¹x = π/2
|x| ≥ 1
5 · Property 4 · Negative Argument
sin⁻¹(−x) = −sin⁻¹(x)
|x| ≤ 1 · Odd function — flip sign
tan⁻¹(−x) = −tan⁻¹(x)
x ∈ ℝ · Odd function
cosec⁻¹(−x) = −cosec⁻¹(x)
|x| ≥ 1 · Odd function
cos⁻¹(−x) = π − cos⁻¹(x)
|x| ≤ 1 · NOT odd — adds π
cot⁻¹(−x) = π − cot⁻¹(x)
x ∈ ℝ · NOT odd — adds π
sec⁻¹(−x) = π − sec⁻¹(x)
|x| ≥ 1 · NOT odd — adds π
★ sin⁻¹, tan⁻¹, cosec⁻¹ → ODD (negate) | cos⁻¹, cot⁻¹, sec⁻¹ → NOT odd (π − value)
6 · Property 5 · Reciprocal Conversions
cosec⁻¹(x) = sin⁻¹(1/x)
|x| ≥ 1 · Convert cosec⁻¹ → sin⁻¹
sec⁻¹(x) = cos⁻¹(1/x)
|x| ≥ 1 · Convert sec⁻¹ → cos⁻¹
cot⁻¹(x) = tan⁻¹(1/x) [x > 0]
Direct reciprocal for positive x
cot⁻¹(x) = π + tan⁻¹(1/x) [x < 0]
Negative x — add π correction
7 · Property 6 · Addition / Subtraction Formulas
tan⁻¹ Addition
tan⁻¹x + tan⁻¹y = tan⁻¹((x+y)/(1−xy))
x>0, y>0, xy < 1 · Standard case
tan⁻¹x + tan⁻¹y = π + tan⁻¹((x+y)/(1−xy))
x>0, y>0, xy > 1 · Add π
tan⁻¹x + tan⁻¹y = π/2
x>0, y>0, xy = 1 · Reciprocals
tan⁻¹x − tan⁻¹y = tan⁻¹((x−y)/(1+xy))
x>0, y>0 · Subtraction
tan⁻¹x + tan⁻¹y + tan⁻¹z = tan⁻¹[(x+y+z−xyz)/(1−xy−yz−zx)]
x,y,z>0, xy+yz+zx < 1
sin⁻¹ Addition
sin⁻¹x + sin⁻¹y = sin⁻¹[x√(1−y²) + y√(1−x²)]
x>0,y>0,(x²+y²)≤1
sin⁻¹x + sin⁻¹y = π − sin⁻¹[x√(1−y²) + y√(1−x²)]
x>0,y>0,(x²+y²)>1
sin⁻¹x − sin⁻¹y = sin⁻¹[x√(1−y²) − y√(1−x²)]
x>0, y>0
cos⁻¹ Addition
cos⁻¹x + cos⁻¹y = cos⁻¹[xy − √(1−x²)√(1−y²)]
x>0, y>0
cos⁻¹x − cos⁻¹y = cos⁻¹(xy+√(1−x²)√(1−y²)) [x≤y]
x,y>0, x≤y
cos⁻¹x − cos⁻¹y = −cos⁻¹(xy+√(1−x²)√(1−y²)) [x>y]
x,y>0, x>y
★ For negative x or y — first apply Property 4 (negate/π correction), THEN use above formulas

9 · Half-Angle ITF Identities
cos⁻¹√((1+x)/2) = sin⁻¹√((1−x)/2)
∀x ∈ [−1,1] · Proved via cos(θ/2) = sin(π/2 − θ/2)
cos⁻¹√((1+x)/2) = (1/2)cos⁻¹x
Half-angle compression
sin⁻¹√((1−x)/2) = (1/2)cos⁻¹x
Alternative form
tan⁻¹√x = (1/2)cos⁻¹((1−x)/(1+x)) x∈[0,1]
Useful domain-range problems
10 · Equation Solving Tricks
sin⁻¹f(x) + cos⁻¹g(x) = π/2 ⟹ f(x)=g(x)
Plus domain check |f(x)|,|g(x)| ≤ 1
sin⁻¹(f) + cos⁻¹(f) = π/2 always
Direct use of Property 3
2tan⁻¹x = sin⁻¹(2x/(1+x²)) [|x|≤1]
Convert 2tan⁻¹ to single ITF
tan⁻¹(tan θ) = θ − nπ [shift to (−π/2,π/2)]
Evaluate out-of-range tan⁻¹(tan x)
cot⁻¹(cot x) = x − nπ [shift to (0,π)]
Evaluate cot⁻¹(cot x)
11 · Inequation Rules [Monotonicity]
x > y ⟺ sin⁻¹x > sin⁻¹y
|x|,|y| ≤ 1 · Increasing
x > y ⟺ cos⁻¹x < cos⁻¹y
|x|,|y| ≤ 1 · Decreasing — flip sign
x > y ⟺ tan⁻¹x > tan⁻¹y
x,y ∈ ℝ · Increasing
12 · Series Summation via Telescoping
tan⁻¹((x−y)/(1+xy)) = tan⁻¹x − tan⁻¹y
Core telescoping identity for series
Σ tan⁻¹(1/(n²−n+1)) = tan⁻¹(n) − tan⁻¹(n−1)
Telescopes → tan⁻¹(∞) = π/2
Σ tan⁻¹(2/n²) = tan⁻¹[(n+1)−(n−1)] / [1+(n−1)(n+1)]
Decompose numerator as difference
Σ cot⁻¹(2ⁿ⁺¹ + 2⁻ⁿ) = π/4
Convert cot⁻¹→tan⁻¹, telescope
★ TRICK: Write T_n = tan⁻¹(f(n)) − tan⁻¹(f(n−1)) → S_n = tan⁻¹(f(n)) − tan⁻¹(f(0))

14 · Cross-Conversion [sin⁻¹↔tan⁻¹↔cos⁻¹]
sin⁻¹(a/b) = tan⁻¹(a/√(b²−a²))
Draw right triangle, get opposite/adjacent
cos⁻¹(a/b) = tan⁻¹(√(b²−a²)/a)
Hypotenuse b, adjacent a
sin(cos⁻¹x) = √(1−x²)
Pythagoras — used in proofs
cos(sin⁻¹x) = √(1−x²)
Same value, different ITF
tan(sin⁻¹x) = x/√(1−x²)
opp/adj from right triangle
cos(tan⁻¹x) = 1/√(1+x²)
sec²θ = 1+tan²θ → cosθ
sin(tan⁻¹x) = x/√(1+x²)
sinθ from right triangle
sec(tan⁻¹x) = √(1+x²)
Used in sec²+cosec² type problems
cosec(cot⁻¹x) = √(1+x²)
sec²(tan⁻¹2)+cosec²(cot⁻¹3)=5+10=15
